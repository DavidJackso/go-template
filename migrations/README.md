# migrations

SQL-миграции через goose.

Формат имени файла: `NNN_описание.sql`  
Формат содержимого:
```sql
-- +goose Up
-- SQL для применения миграции

-- +goose Down
-- SQL для отката
```

`embed.go` компилирует все `.sql` файлы в бинарь — миграции не нужно деплоить отдельно.

Применение: `go run ./cmd/migrate`
