# internal/app

Корень зависимостей (DI). Создаёт и связывает все компоненты: DB pool, JWT manager, репозитории, сервисы, HTTP сервер.

`New` — собирает граф зависимостей, возвращает `*App`.  
`Run` — запускает `http.Server`, блокирует до сигнала или ошибки.  
`Close` — освобождает ресурсы (закрывает DB pool).

Когда добавляешь новый сервис или адаптер — инициализируй его здесь и передай туда, где нужен.
