# internal/config

Загружает конфиг из `config.yaml` (или `./config/config.yaml`) и env переменных с префиксом `APP_`.

Env перекрывает файл. Файл необязателен — если не найден, работает только на env/дефолтах.

`db.dsn` и `jwt.secret` обязательны — без них `Load()` вернёт ошибку.

Маппинг env → yaml: точки заменяются на `_`. Пример: `APP_JWT_TTL` → `jwt.ttl`.
